                                    9388              * this file is a single user path and is used to update space
                                    9389              * used, control various batch limits and contain the users
                                    9390              * password.  it is also used to store cross session information
                                    9391              * of which the best example is the flag that indicates that an
                                    9392              * execfile is to be automatically executed when the user logs
                                    9393              * on.

                                    9395              * the kwio routine obtains the kwrio resource and of course
                                    9396              * waits if it is not available.  when the resource is obtained
                                    9397              * the I/O is started and the user waits until it completes
                                    9398              * upon completion of the I/O the kiopost routine is entered
                                    9399              * this routine simply queues the kwrio user JCB to the setup
                                    9400              * queue so that he can be dispatched again.  when the user
                                    9401              * is dispatched he will free up the resource and that will

                                    9406              *      R1  -  address of the password record.
